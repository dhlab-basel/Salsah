App VIEW¶

The Salsah app has only a few main view components. They’re following the routing setup in app-routing.module.ts (with some exceptions e.g. /search). We organize the main view components in the folders called dashboard, documentation and login. The last sub-folder in the view is for every other module, which should be reusable in all the main components.
